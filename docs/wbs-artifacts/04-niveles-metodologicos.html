El sistema operativo del proyecto

AtaraxiaDive no se entiende solo como una WBS ni solo como un backlog. Funciona con una cadena de niveles: arriba se decide propósito y valor; abajo el skill real /implement-us orquesta una US mediante agentes especializados. La vista ahora usa las fases reales declaradas en .claude/skills/implement-us/skill.md.
