# Manual de Usuario — AtaraxiaDive

AtaraxiaDive es una plataforma para la gestión de torneos de apnea (freediving). Según tu rol en el deporte, este manual tiene una sección dedicada para vos.

---

## ¿Por dónde empezar?

**Si querés ver torneos o resultados sin registrarte** → [Portal Público](portal-publico/index.md)

**Si organizás torneos** → [Portal Organizador](portal-organizador/index.md)

**Si competís** → [Portal Atleta](portal-atleta/index.md)

**Si arbitrás competencias** → [Portal Juez](portal-juez/index.md)

---

## Acceso a la plataforma

La plataforma está disponible en **[ataraxiadive.fly.dev](https://ataraxiadive.fly.dev)**. No es necesario instalar nada — funciona desde el navegador de tu celular o computadora.

La pantalla de inicio muestra directamente el listado de torneos disponibles, sin necesidad de iniciar sesión.
